# Cocktail finder | NextDots Challengue | RN 0.61.5

## Getting started

Run `yarn` inside proyect folder.

## For IOS (only work on MAC device) you need to run

- `cd ios`
- `pod install`

## Launch Android

`react-native run-android`

## Launch iOS

`react-native run-ios`

## VSCode Extensions

- Prettier
- TSLint
- ESLint
